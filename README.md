## 🔧 安裝步驟
1. node.js 版本需為 v22 或以上，請參考 [Node.js 官方網站](https://nodejs.org/) 下載安裝。
2. 安裝 [yarn](https://yarnpkg.com/)
3. 安裝 (可選) [nvm](https://github.com/coreybutler/nvm-windows)
4. 安裝相關依賴
[discord.js]
[fast-glob]
[vite-node]
[dotenv]
[pinia]
[vue]

```bash
yarn app discord.js fast-glob vite-node dotenv pinia vue
```
